# [Mod] X Obsidian-Mese [x_obsidianmese]

This MOD for Minetest adds more use for Mese and Obsidian. Add more variety of tools and blocks.

![screenshot](screenshot.1.png)

## Dependencies

- default

## Optional Dependencies

- tnt
- x_farming
